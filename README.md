# ToDoList App V2

Aplikasi ToDoList modern dibuat dengan **React** dan **Tailwind CSS**.  
Fitur utamanya: menambahkan jadwal (schedule) berdasarkan tanggal, dan menambahkan tugas (task) di dalam jadwal tersebut.
